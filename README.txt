# BoiNosAres - Jogo em Python

Este é um jogo feito em Python usando a biblioteca pygame.

## Como rodar o projeto:

1. (Opcional) Crie e ative um ambiente virtual:
   python -m venv .venv

   No Windows:
   .\.venv\Scripts\activate

   No Linux/macOS:
   source .venv/bin/activate

2. Instale as dependências necessárias:
   pip install -r requirements.txt

3. Execute o jogo com:
   python code/main.py

## Requisitos

- pygame (incluso no requirements.txt)

## Como jogar

Use o "ESPAÇO" para fazer o player voar. Desvie dos obstáculos.
